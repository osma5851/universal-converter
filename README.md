# Universal Unit Converter

A beautiful, modern unit converter built with Next.js, TypeScript, and Tailwind CSS.

## Features

- **10 Unit Categories**: Length, Weight, Volume, Temperature, Area, Speed, Time, Digital Storage, Pressure, and Energy
- **90+ Units**: Comprehensive coverage of metric and imperial units
- **Real-time Conversion**: Instant results as you type
- **Swap Units**: Quick swap button to reverse conversions
- **Copy to Clipboard**: Easy copying of results
- **Beautiful UI**: Light theme with smooth animations
- **Fully Responsive**: Works on all devices

## Categories

| Category | Units Available |
|----------|-----------------|
| Length | Meter, Kilometer, Centimeter, Mile, Yard, Foot, Inch, etc. |
| Weight | Kilogram, Gram, Pound, Ounce, Stone, Metric Ton |
| Volume | Liter, Milliliter, Gallon, Cup, Fluid Ounce, etc. |
| Temperature | Celsius, Fahrenheit, Kelvin |
| Area | Square Meter, Hectare, Acre, Square Foot, etc. |
| Speed | m/s, km/h, mph, Knot |
| Time | Second, Minute, Hour, Day, Week, Month, Year |
| Digital Storage | Byte, KB, MB, GB, TB, PB |
| Pressure | Pascal, Bar, PSI, Atmosphere |
| Energy | Joule, Calorie, kWh, BTU |

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Tech Stack

- [Next.js 14](https://nextjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
